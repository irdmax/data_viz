# Data_Viz

## Viewing the Biennale Poster Image

This repository contains a visualization of the "Global Expansion of Biennales" poster.

### Option 1: View the extracted world map image
The world map image has been extracted from the HTML file and saved as:
- **world_map.png** - A PNG image (3200 x 1288 pixels) showing the world map

Simply open `world_map.png` in any image viewer to see the map.

### Option 2: View the full poster in a web browser
Open `biennale_poster_source.html` in a web browser to see the complete poster with all visualizations and text.

### Option 3: Extract the image yourself
Run the extraction script:
```bash
python3 extract_image.py
```

This will extract the base64-encoded image from the HTML file and save it as `world_map.png`.
